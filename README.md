## 📖 Sobre o Projeto

O Atlas é uma plataforma de bem-estar desenvolvida para conectar pessoas a experiências de saúde, qualidade de vida e desenvolvimento humano.

Nosso objetivo é democratizar o acesso ao bem-estar por meio da tecnologia.
